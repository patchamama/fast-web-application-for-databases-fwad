# Fast Web Application for Databases (FWAD)

# Develop

### Install php (Mac OsX)

```
brew tap shivammathur/php
brew install shivammathur/php/php@5.6

# point php to the 5.6 install
brew link --overwrite --force shivammathur/php/php@5.6
php --version
```
